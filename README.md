### 简单使用Vue-CLI（脚手架）搭建一个Vue的项目
1. 使用npm去安装Vue-CLI 
` npm install -g @vue/cli `
2. 创建一个项目
` vue create vuetest (项目名) `
就可以开发了
开发好了可以使用npm的命令去解释运行
` npm run serve `
还可以工程打包、分发，运行在客户端哦
` npm run build `

### Vue 使用饿了么UI组件
1. npm安装好饿了么的组件
2. ` vue add element `
3. 就可以使用UI组件了  

### Vue 使用iveiw UI组件
1. npm安装好饿了么的组件
2. ` vue add iveiw `
3. 就可以使用iveiw UI组件了
